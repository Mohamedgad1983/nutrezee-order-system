## Cross-module contracts (the only allowed inter-module calls)

| Caller → Owner | API used | Purpose |
|---|---|---|
| M01/M02 → M04 | PhoneMatch, profile read, allergy read | Match, dedup, chain step 2 |
| M01/M02/M03 → M05 | Catalog read, AllergenResolver, RoutingRule read (M08 too) | Validation, freezing, routing |
| M02 → M03 | OrderFactory.createFromDraft | The approval conversion (single entry point) |
| M03 → engine (M16 config) | TransitionService | All status changes |
| M07 → M03 | payment status events → gate check | DEC-009 gate |
| M08 → M03 | day rollup transitions | READY_TO_PACK/PACKED |
| M19 → M04/M05/M03 | Import APIs (not raw tables) | ADR-010 even for imports |
| All → M13/M14/M16 | AccessDecision, AuditWriter, Settings | Foundation |

Everything else: events via outbox only. Any new cross-module call requires a spec amendment here.
