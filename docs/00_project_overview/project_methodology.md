# Nutrezee Project Methodology

This project must follow the approved delivery methodology:

0. Safety / Backup / Staging
1. Discovery
2. Context Engineering
3. Workflow Design
4. Architecture
5. Gap Analysis
6. Implementation Plan
7. Development
8. Testing
9. Release

Rules:
- Do not touch production directly.
- Do not start coding before discovery and gap analysis.
- Use separate agents and worktrees.
- Store credentials only in environment variables.
- Review before merge or release.
